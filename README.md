# Dice Roller
A test of the Vue.js framework

## Setup

- Clone this
- Go to `/diceroller` folder
- Run `npm install`
- Run `npm run dev`
- Roll dem dice!

## Tools used

- Vue.js 2.1.10
- Webpack 2.2.1
- SASS

Written in [JavaScript Standard Style](https://github.com/feross/standard/blob/master/RULES.md#javascript-standard-style).
